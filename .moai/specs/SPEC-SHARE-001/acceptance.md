# SPEC-SHARE-001 수락 기준

## 테스트 시나리오

### AC-SHARE-001: 공유 버튼 탭 시 카메라/갤러리 실행

**Given** 사용자가 모바일 브라우저에서 청첩장 페이지를 열고 있고
**And** "청첩장 공유하기" 섹션이 화면에 보이는 상태일 때
**When** "사진 찍고 공유하기" 버튼을 탭하면
**Then** 기기의 카메라 앱 또는 갤러리 선택 화면이 실행된다
**And** 사용자는 사진 촬영 또는 기존 사진 선택이 가능하다

### AC-SHARE-002: 사진 촬영/선택 완료 시 미리보기 모달 표시

**Given** 카메라 앱 또는 갤러리가 실행된 상태에서
**When** 사용자가 사진을 촬영하거나 갤러리에서 사진을 선택하면
**Then** 리사이징된 이미지가 미리보기 모달에 표시된다
**And** 배경 페이지의 스크롤이 잠긴다
**And** 모달 하단에 "공유하기" 버튼이 표시된다
**And** 모달 상단에 "닫기(X)" 버튼이 표시된다

### AC-SHARE-003: Web Share API 환경에서 시스템 공유 시트 표시

**Given** Web Share API를 지원하는 브라우저(iOS Safari 15+, Android Chrome 75+)에서
**And** 미리보기 모달이 열린 상태일 때
**When** "공유하기" 버튼을 탭하면
**Then** 시스템 공유 시트가 표시된다
**And** 공유 시트에서 카카오톡을 포함한 앱 목록을 선택할 수 있다

### AC-SHARE-004: KakaoTalk 인앱 브라우저에서 Kakao SDK 폴백 표시

**Given** KakaoTalk 인앱 브라우저(Web Share API 미지원)에서 청첩장을 열고 있고
**And** 미리보기 모달이 열린 상태일 때
**When** 사용자가 모달을 확인하면
**Then** Web Share API 버튼 대신 "카카오톡으로 공유" 버튼이 표시된다
**And** 해당 버튼을 탭하면 카카오톡 피드 메시지 공유가 실행된다
**And** 피드 메시지에 청첩장 URL과 OG 이미지가 포함된다

### AC-SHARE-005: 공유 시트에서 취소 시 에러 없이 복귀

**Given** Web Share API를 통해 시스템 공유 시트가 표시된 상태에서
**When** 사용자가 공유를 취소하면 (뒤로가기, X 버튼 등)
**Then** 에러 메시지 없이 미리보기 모달로 복귀한다
**And** 미리보기 이미지와 버튼이 정상 상태로 유지된다
**And** 다시 "공유하기" 버튼을 탭할 수 있다

### AC-SHARE-006: Kakao SDK 실패 시 URL 클립보드 복사 + 토스트

**Given** Kakao SDK 로딩 실패 또는 공유 호출 실패 상태에서
**When** 시스템이 Kakao 공유 실패를 감지하면
**Then** 청첩장 URL(`https://songhongseob.github.io`)이 클립보드에 복사된다
**And** "링크가 복사되었습니다" 토스트 메시지가 화면 하단에 3초간 표시된다

## 엣지 케이스 시나리오

### EC-001: 카메라 권한 거부

**Given** 사용자가 브라우저의 카메라 권한을 거부한 상태에서
**When** "사진 찍고 공유하기" 버튼을 탭하면
**Then** "카메라에 접근할 수 없습니다" 토스트 메시지가 3초간 표시된다
**And** 미리보기 모달은 열리지 않는다

### EC-002: 대용량 이미지 파일 (10MB+)

**Given** 사용자가 10MB 이상의 고해상도 사진을 선택했을 때
**When** 이미지 리사이징이 수행되면
**Then** Canvas API를 통해 최대 1920px, JPEG 0.85로 압축된다
**And** 리사이징 완료 후 미리보기 모달에 정상 표시된다
**And** 리사이징 중 UI가 블로킹되지 않아야 한다 (로딩 인디케이터 표시)

### EC-003: 비이미지 파일 선택 시도

**Given** 갤러리에서 비이미지 파일(동영상 등)이 선택된 경우
**When** 파일 change 이벤트가 발생하면
**Then** `file.type`이 `image/*`에 해당하지 않으면 무시한다
**And** 토스트로 "이미지 파일만 선택해주세요" 안내한다

### EC-004: 오프라인 상태에서 Kakao SDK 로딩 실패

**Given** 네트워크 연결이 불안정하여 Kakao SDK 로딩이 실패한 경우
**When** Kakao Share 폴백이 필요한 상황에서
**Then** `typeof Kakao === 'undefined'` 체크 후 클립보드 복사 폴백으로 전환한다
**And** "링크가 복사되었습니다" 토스트를 표시한다

### EC-005: 모달 열린 상태에서 뒤로가기

**Given** 미리보기 모달이 열린 상태에서
**When** 브라우저 뒤로가기 버튼을 누르면
**Then** 모달이 닫히고 scroll lock이 해제된다
**And** 페이지가 이전 상태로 복귀한다 (history.pushState 활용)

### EC-006: 데스크톱 브라우저에서 접근

**Given** 데스크톱 브라우저에서 청첩장 페이지를 열었을 때
**When** "사진 찍고 공유하기" 버튼을 클릭하면
**Then** 파일 선택 다이얼로그가 열린다 (capture 속성은 무시됨)
**And** 이후 공유 흐름은 모바일과 동일하게 동작한다
